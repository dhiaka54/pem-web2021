Selektor universal adalah selektor yang digunakan unutk menyeleksi semua elemen pada jangkauan (scope) tertentu. Contoh:

* { border: 1px solid grey; }

Artinya semua elemen akan memiliki garis solid dengan ukuran 1px dan berwarna grey. Selektor universal bisanya digunakan untuk me-reset CSS. Pada halaman HTML, ada beberapa CSS bawaan browser seperti padding dan margin pada elemen tertentu. Reset bertujuan untuk menghilangkan padding dan margin tersebut. Contoh CSS reset:

* { padding: 0; margin: 0; } Maka semua elemen tidak akan memiliki padding dan margin.
